## Stadion bron ilovasi

Bu repo 2 qismdan iborat:

- `backend/` — Node.js + Express + MongoDB API (Railway’da deploy qilish uchun)
- `mobile/` — Flutter ilova (Android + iOS)

### Backend

1) `backend/.env.example` ni `backend/.env` ga ko‘chiring va qiymatlarni to‘ldiring.

2) Ishga tushirish:

```bash
cd backend
npm i
npm run dev
```

### Mobile (Flutter)

Flutter loyihasi keyingi bosqichda qo‘shiladi.
